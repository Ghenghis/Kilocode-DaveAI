## 🔧 Installation

### Quick Start
```bash
# Install CLI
npm install -g @kilocode/cli

# Run validation
node scripts/validate-release.js

# Start development
bun run dev
```

### VS Code Extension
1. Open VS Code
2. Search for "Kilo Code" in Extensions Marketplace
3. Click "Install"
4. Follow setup instructions
